# Zincir

![Badge](https://hitscounter.dev/api/hit?url=https%3A%2F%2Fgithub.com%2Fmuammergns%2FZincir%2F&label=Visit&icon=award&color=%23198754&message=&style=for-the-badge&tz=Asia%2FIstanbul)


**Zincir**, .NET 10 ve Avalonia UI ile geliştirilen, cross platform verimlilik uygulamasıdır.

### Hedefler
- **Alışkanlık Takibi:** Zinciri kırma felsefesiyle sürdürülebilir alışkanlıklar.
- **Görev Yönetimi:** Eisenhower matrisi tabanlı akıllı yapılacaklar listesi.
- **Odaklanma:** Todo ve Habit entegreli Pomodoro zamanlayıcı.

### Durum
Proje henüz (Pre-alpha) geliştirme aşamasının başındadır.
